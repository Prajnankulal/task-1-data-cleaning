Data Cleaning and Preprocessing Task
Objective

To clean and prepare the Titanic dataset by handling missing values, duplicates, and inconsistent formats.

Dataset

Titanic Dataset (Seaborn)

Steps Performed

Checked and handled missing values using fillna()
Removed duplicate records using drop_duplicates()
Dropped column with excessive missing values (deck)
Standardized column names (lowercase, no spaces)

Tools Used

Python (Pandas)
Seaborn
VS Code / Jupyter Notebook

Output

Cleaned dataset (cleaned_titanic.csv) ready for analysis.
